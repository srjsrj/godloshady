Год лошади

Конюшня, которая построена для частных владельцев лошадей и просто для людей, которым нравится это прекрасное животное. Конюшня располагается в Красноярском крае, между деревнями Замятино и Арей. Занятия проводятся под постоянным контролем инструктора, на огороженной территории клуба, в удобное для вас время. В вечернее время плац освещен. Для маленьких детей занятия проводятся на пони. Для опытных всадников возможны прогулки по живописным окрестностям.
